# grey10k

One of the nine shades of grey included. The lower the k-value, the
lighter the shade. grey10k is the lightest grey. hex value: "#E6E7E8".
Useful for light reference lines, boxes or other areas.

## Usage

``` r
grey10k
```

## Format

An object of class `character` of length 1.
