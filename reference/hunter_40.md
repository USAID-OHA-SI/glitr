# hunter @ 40% tint

One of the five shades of hunter. The number is a percent of the
original color, ranging from 100 (100%) down to 20 (20%) at 20%
intervals. The lower the number, the lighter the tint of the original
color.

## Usage

``` r
hunter_40
```

## Format

An object of class `character` of length 1.
